⚙ 服务器设置

⚙ 基本设置
📁 配置文件参考
⚠ 修改并重启后浏览器将自动跳转到新地址。
环境变量
变量默认值说明
DRUIDCLAW_TOKENccWeb 访问密码（--passwd 可覆盖）
DRUIDCLAW_RUN_DIR./run运行时数据目录（所有 JSON 配置文件所在位置）
DRUIDCLAW_WEB_HOST0.0.0.0绑定地址（启动时写入，优先于 config.json）
DRUIDCLAW_WEB_PORT19123监听端口（启动时写入，优先于 config.json）
启动参数 python -m druidclaw.web / start.sh
参数默认值说明
--host读 config.json绑定地址，覆盖已保存配置
--port读 config.json监听端口，覆盖已保存配置
--passwd读 DRUIDCLAW_TOKEN访问密码，覆盖 DRUIDCLAW_TOKEN 环境变量
--workdir.新建会话的默认工作目录
--reloadfalse代码变更时自动重载（开发用）
运行时配置文件 $DRUIDCLAW_RUN_DIR/
文件用途关键字段
config.jsonWeb 服务器绑定地址/端口host（默认 0.0.0.0）、port（默认 19123）
cards.json所有 Card 配置，重启后自动恢复见下方 Card 字段说明
feishu.json飞书遗留全局凭证（兼容旧版）app_id、app_secret
feishu_bridge.jsonIM Bridge 通用配置reply_delay（默认 2.0 秒）
tasks.json定时任务列表name、session、prompt、interval / cron
ssh_history.jsonSSH 连接历史（不含密码）host、port、username、key_path
cards.json — Card 字段说明
字段适用类型说明
id全部自动生成的 8 位 hex ID
type全部claude / feishu / telegram / dingtalk / qq / wework
name全部显示名称
workdir全部工作目录（默认 .）
auto_start全部启动时自动运行（默认 true）
auto_approveIM 类自动通过权限提示，跳过确认（默认 false）
reply_delayIM 类回复延迟秒数，范围 0.5–60（默认 2.0）
argsclaude附加 Claude 启动参数数组
app_id / app_secretfeishu / dingtalk飞书 App ID + Secret；钉钉 App Key + Secret
tokentelegramTelegram Bot Token
ws_url / access_tokenqqOneBot v11 WebSocket 地址及可选鉴权 Token
corp_id / agent_id / corp_secretwework企业微信企业 ID、应用 ID、应用 Secret
wework_token / encoding_aes_keywework消息校验 Token 及 43 位 AES Key
其他运行时路径
路径说明
./run/logs/PTY 会话日志（.log 文本 / .raw 二进制录像）
./run/daemon.sockUnix socket（daemon 模式 IPC）
./run/daemon.piddaemon 进程 PID 文件
~/.claude/skills/*/SKILL.mdSkills 侧边栏数据来源
./run/logs/ssh_{name}_{ts}.logSSH 会话临时日志
